A:

It seems the rplidar_ros package cannot bind the /dev/ttyUSB0 port:
RPLIDAR running on ROS package rplidar_ros
SDK Version: 1.5.7
Error, cannot bind to the specified serial port /dev/ttyUSB0.

Check if you have permission to read/write serial ports (your user must be in "dialout" group), or if the sensor has been assigned to another port (e.g., /dev/ttyUSB1).
This question might also help you.
