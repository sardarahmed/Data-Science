Q:

Is it possible to install ros diamondback in ubuntu 12.04

Hi,
I wonder if it is possible to install the ros diamondback in ubuntu 12.04. I have already seen the installation page, http://wiki.ros.org/diamondback/Installation/Ubuntu, but it seems that it doesn't have the setup for ubuntu 12.04. what should i do? thanks for any help.

A:

ROS Diamondback was released before 12.04, and was not supported on 12.04.
ROS Diamondback is no longer supported. If you're doing a new installation of ROS, you should seriously consider using a newer, supported version of ROS.
