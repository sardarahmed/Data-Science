Q:

Where does the parameter server store data?

Where on disk does the parameter server store data to when it's not running?

Originally posted by Cerin on ROS Answers with karma: 940 on 2018-02-02
Post score: 0

A:

Nowhere. The parameter server comes up with the rosmaster. Once you shut that down, the parameter server shuts down as well and all parameter info is lost.
