Q:

ARtoolkit - rviz

How can I use the ARtoolkit library with rviz?
(Ubuntu 10.04, ROS electric)

Originally posted by Janina on ROS Answers with karma: 11 on 2012-05-13
Post score: 0

A:

You can use the package ar_pose. It runs the ARToolkit marker detection and publishes both detection messages and transforms that can easily be visualized in rviz. See the package wiki page for screenshots and a demo video.
