Q:

Save nav_msgs/OccupancyGrid as png

Hi,
map_server saves an occupancy grid as a pgm, but is there a way to save the map as a png?

A:

You can convert the pgm to a png with gimp, photoshop, or the convert command-line tool. (sudo apt-get install imagemagick ;  convert map.pgm map.png )
