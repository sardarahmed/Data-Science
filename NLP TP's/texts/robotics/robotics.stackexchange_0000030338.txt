Q:

malloc assertion in move_base

I get the following error running move_base against diamondback:
move_base: malloc.c:4630: _int_malloc: Assertion
`(unsigned long)(size) >= (unsigned long)(nb)' failed.
[move_base_node-1] process has died [pid 2403, exit code -6].

A:

The cause of the error was that the nav launch file loaded a plugin (sbpl_recovery) that was not part of the released .deb packages, and had to be recompiled manually.
