A:

That sounds like you have a newer version of rosinstall on your system than vcstools.  Please make sure both vcstools and rosinstall are up to date.
If you are on ubuntu I suggest uninstalling any pip based installation and using apt-get install python-rosinstall python-vcstools
Otherwise use the -U option to pip to update.
