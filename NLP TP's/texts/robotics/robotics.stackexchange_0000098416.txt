I have only one point cloud map in autoware，Can I do global path planning by using op_global_planner.I don't want to use waypoint_maker

A:

You can't do that with only point cloud, you need to create a Road Network map on top of the point cloud.
for op_global_planner to work, you need either vector map or km map.
To create vector map you can use TierIV tools
To create KML map you can use ASSURE mapping tool.
