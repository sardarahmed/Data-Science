Q:

how can i change "minimum_waypoint_count"

how can i change default value of "minimum_waypoint_count" or other parameters
"rosparam set /move_group/ompl/minimum_waypoint_count 100" has no effect.

Originally posted by omeranar1 on ROS Answers with karma: 31 on 2022-04-25
Post score: 0

A:

in "move_group.launch" file
<param name="planning_scene_monitor/publish_planning_scene_hz" value="100.0" />
<param name="ompl/minimum_waypoint_count" value="20.0" />
