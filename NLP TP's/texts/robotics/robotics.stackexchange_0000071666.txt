Q:

Warning using the Package Robot_localization

Hi everybody,
Just a question, Is it normal the next WARNING using the package Robot_localization? :
WARNING The following node subscriptions are unconnected:

/ekf_localization:

/set_pose

Regards

Post score: 0

A:

'ekf_localization' is subscribed to '/set_pose', but nothing is publishing it. I think that's a ROS-level warning, and doesn't originate within 'ekf_localization_node' itself.
