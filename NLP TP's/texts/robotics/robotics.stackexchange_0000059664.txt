A:

On none of the four.
Quote @tfoote from his mail to ROS Users:

To make sure that we have adequate testing we will delay the final release into June.

Edit: Yes you can install Beta 1 and use it.
You should receive the updates for the stable version with sudo apt-get upgrade.
