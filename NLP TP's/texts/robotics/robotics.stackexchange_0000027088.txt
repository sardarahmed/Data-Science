A:

Sounds like you are missing an installation from packages and a try to compile gazebo from scratch. If this is your case, please be sure to clean up first all gazebo stuff from your system (find /usr -name gazebo could help to detect what needs to be cleaned up).
Once this is done, proceed with a simple:
apt-get install -f 

That was in your log and probably the package manager was not in a good state. And after fixing the errors, try again the installation.
I was able to launch the apt-get install gazebo3 into a precise system without any problem so I'm assuming that the problem could be in your system.
Thanks.
