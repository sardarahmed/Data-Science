Q:

ps3joy in Fuerte

I think the title says it all, since it does not appear to be in the archive any longer, what's the best way to get the ps3joy.py node spun up in Fuerte?

Post score: 1

A:

Install: # apt-get install ros-fuerte-joystick-drivers
Run: # rosrun ps3joy ps3joy.py
