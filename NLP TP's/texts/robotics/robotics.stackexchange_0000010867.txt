Q:

Kuka KR-C4 Documentation

Is there an official documentation from Kuka that explains at a beginner level how to start programming Kuka Robots equipped with KR-C4 controllers?

A:

The Kuka KRC4 Controller can be programmed using the KRL language. This official Kuka manual  gives a good introduction to both basics of hardware and software.
KRL is a high level programming language which runs on a mixed Windows-VxWorks environment.
