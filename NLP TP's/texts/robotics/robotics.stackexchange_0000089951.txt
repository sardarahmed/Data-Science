Q:

changing system of turtlebot from indigo to kinetic

I need to change the system  of the robot from Ubuntu 14.04 to Ubuntu 16.04

A:

If you just have changed from ros indigo to kinetic, you can install turtlebot package for kinetic using:
sudo apt-get install ros-kinetic-turtlebot
