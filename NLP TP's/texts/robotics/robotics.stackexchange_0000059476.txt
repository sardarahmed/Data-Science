Q:

Gmapping file map output structure

Hello everybody! Where can I find the map 2d file structure of the output of mapserver (gmapping)?

A:

Gmapping publishes a /map topic, normally you would use the map_server package to save this topic to a file:
rosrun map_server map_saver

See the wiki page for other options http://wiki.ros.org/map_server
