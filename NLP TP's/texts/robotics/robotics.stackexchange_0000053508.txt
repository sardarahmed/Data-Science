A:

Usando um simulador como Stage, você pode gerar um grande número de robôs. Estes robôs podem ser controlados através de ROS. Supondo que você tenha o código que você gostaria que cada robô executar, então esta é uma maneira que você pode demonstrar robótica de enxame.
Translation:
Using a simulator like Stage, you can generate large numbers of robots. These robots can be controlled through ROS. Assuming that you have the code that you would like each robot to run, then this is one way that you can demonstrate swarm robotics.
