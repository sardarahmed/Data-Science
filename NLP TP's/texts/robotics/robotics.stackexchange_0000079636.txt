turtlebot urdf.xacro file for VLP-16 3D sensor

I am trying to add new sensor Velodyne VLP16  for the turtlebot as given in http://wiki.ros.org/turtlebot/Tutorials/indigo/Adding%20New%203D%20Sensor.
How to generate a urdf xacro file or where to find ?

A:

You can find the URDF description of VLP16 in the package velodyne_description.
Here is the direct link to the VLP-16.urdf.xacro file: https://bitbucket.org/DataspeedInc/velodyne_simulator/raw/b2b4496e8150c4e4082a0b8227922a721f4dc91d/velodyne_description/urdf/VLP-16.urdf.xacro
Good luck!
