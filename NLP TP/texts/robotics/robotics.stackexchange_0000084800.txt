A:

If you look carefully through the output from catkin_make you can see that the first error reported is this:
In file included from /home/icrs/manipulator_proj/src/dynamixel_control/src/dynamixel_control.cpp:41:0: /home/icrs/manipulator_proj/src/dynamixel_control/src/dynamixel_control.h:19:90: fatal error: dynamixel_sdk.h: No such file or directory compilation terminated.

The compiler cannot find the dynamixel_sdk.h file, which is included from file /home/icrs/manipulator_proj/src/dynamixel_control/src/dynamixel_control.h
Update: Okay Error 2 is a bit more complicated, you're trying to print out the joint angle to the screen so printf is expecting an integer (you probably want to change this if %f for a floating point instead, but that's not the problem here). But you're passing it a pointer to a subscriber object, this is a way off the code you need to do this. I recommend starting a new question and posting the source code you've attempted to get this working with, then we'll be able to work out what you need to do.
