Comment by Ivan Rojas Jofre on 2012-02-26:
If you install ROS by Synaptic, then you can uninstall ROS with Synaptic.
If you install ROS by terminal, then you execute this instrucction: "sudo apt-get remove ros"
Comment by joq on 2012-02-27:

A:

If you installed with apt:
sudo apt-get remove ros

If you installed with svn:
rm -rf /path/to/your/ros/directory
