Q:

ROS network is disabled

When I run the node_manager, on the top left, "ROS Network [disabled]" is shown.

A:

Run master_discovery node if you have more than 1 host, $ rosrun master_discovery_fkie master_discovery.
If not, do not need to start master_discovery.
You can start master_discovery by pressing on "start" button.
