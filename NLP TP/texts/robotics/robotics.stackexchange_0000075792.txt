Q:

Can Gmapping package in ROS can create map at Runtime

We have gmapping package in ROS
Can we create a map in ROS at runtime using gmapping package and without using rosbag file ?
If yes ,then how it can be done ?

Originally posted by Atul Divekar on ROS Answers with karma: 26 on 2016-08-14
Post score: 0

A:

Yes this can be done.
Just set up gmapping as you do with the bagfile, but use the inputs of the real robot instead of the topics coming from the bagfile (i.e. the laserscanner message and the tf)
