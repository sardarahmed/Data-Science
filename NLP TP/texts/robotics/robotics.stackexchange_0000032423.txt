Can I use teleop in p2os system with only one computer instead a computer emisor and a computer receptor?

A:

That should be possible. You'll want to run everything described in the tutorial on the computer on the robot, and you'll have to attach your joystick to computer on the robot.
You shouldn't have to set ROS_MASTER_URI since all of your ros nodes are running on the same computer.
