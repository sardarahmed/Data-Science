A:

when I try to run the command apt-get install ros-hydro-industrial-desktop it appears there is no package available

The Hydro packages are not available on Kinetic, no.
There is in fact no release of industrial_desktop for Kinetic. The only way to use the industrial packages would be to build them from source in a catkin workspace.
