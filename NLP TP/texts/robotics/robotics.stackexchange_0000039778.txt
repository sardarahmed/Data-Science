A:

Are you installing on Electric or Fuerte? rosjava_jni has not been released yet for Fuerte, there you need to install from source (repository url: git://code.in.tum.de/git/client-rosjava). In Electric, do:
sudo apt-get install ros-electric-client-rosjava-jni
