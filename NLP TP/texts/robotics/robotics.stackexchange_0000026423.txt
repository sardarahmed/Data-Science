Q:

How to get the value of "_index" used in gazebo::physics::Joint Class

In class "Joint",lots of  Public Member Functions need the "index" as a Parameter.How can I get its value?

Originally posted by Sway on Gazebo Answers with karma: 3 on 2013-05-23
Post score: 0

A:

Because the joint connects two links, one being the child and the other the parent, this can be (0, 1), where index = 0 it is the child link and index = 1 it is the parent link.
