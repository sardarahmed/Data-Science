A:

Turtlebot3 can be installed directly from apt: sudo apt install ros-noetic-turtlebot3.
Check the emanual: https://emanual.robotis.com/docs/en/platform/turtlebot3/quick-start/#pc-setup for more information.
For simulations: sudo apt install ros-noetic-turtlebot3-simulations
