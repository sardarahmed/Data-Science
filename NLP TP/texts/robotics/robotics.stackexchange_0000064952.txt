A:

Since you're "very new" to ROS, you likely don't need foreign_relay. ROS is a distributed system that supports topic sharing between computers out-of-the-box -- you simply need to use the ROS_MASTER_URI environment variable (see the Running ROS across multiple machines tutorial).
