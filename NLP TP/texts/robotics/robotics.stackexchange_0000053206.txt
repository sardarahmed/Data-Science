Q:

launch file to make rviz to subscribe to laserscan published from a bag file

how can I make rviz to display laserscan data played from a bag file using a launch file?

A:

Hi Solmaz,
you should first run everything and set that up as you like. Once your rviz looks just as you want, save the configuration file of rviz and pass it with argument '-d' to your rviz node instance in your launchfile. This way, your rviz will look always the same.
