A:

Hey,
The problem is that it is linking the wrong pcl library. pcd_converter is looking for getMinMax3D and it is looking in pcl/filters/voxel_grid.h. The function that takes a point cloud and two points is in pcl/common/common.h
to fix this problem edit include/pcd_converter.h and add:
#include <pcl/common/common.h>
then run cmake again and  it should build correctly
hope this helps!
