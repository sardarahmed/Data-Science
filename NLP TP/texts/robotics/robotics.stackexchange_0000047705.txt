Q:

pointcloud_to_laser output value is constant

I am using pointcloud_to_laser package in fuerte, the output laser range value is constant but when i use the same code in different laptop(electric) it works fine. Does anyone has a possible explanation for this behavior??

A:

The problem was that I was using kinect.launch, launch file in turtlebot_bringup package. When I was calling this file I was not getting any pointcloud, but when I used openni.launch(openni_launch), kinect worked fine.
