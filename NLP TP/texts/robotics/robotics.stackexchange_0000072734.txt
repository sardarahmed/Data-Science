Q:

different between cd and roscd?

Hi,I'm a new learner. Currently learn use Linux and ROS basic command. So what is the different between cd and roscd ?

Originally posted by Zero on ROS Answers with karma: 104 on 2016-03-13
Post score: 1

A:

cd is the standard linux command. roscd however also knows about the installed ros packages, so you can go to a package without having to type (or even know) the exact path. Just try 'roscd you_package'
