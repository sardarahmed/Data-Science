Have you had a partially built moveit package in your own /root/catkin_ws directory earlier?
If so,
there is no need to install from source that usually causes problem for freshmen like me.

Follow the steps:
$ cd ~/catkin_ws %YOUR_WORK_PLACE%
$ rm -rf devel build
$ catkin_make
$ source ~/.bashrc
