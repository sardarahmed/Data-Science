Q:

Releasing RobotWebTools on CDN

What is the best method for getting a project included in the cdn associated with Robot Web Tools?
a la http://cdn.robotwebtools.org/roslibjs/current/roslib.min.js

Originally posted by David Lu on ROS Answers with karma: 10932 on 2015-02-23
Post score: 0

A:

The CDN is managed by an internal AWS account that the developers have access to. At this time, we haven't allowed for a way to publish arbitrary libraries due to a number of reasons. If you'd like to publish there, you will need to contact a developer directly to upload the files for you!
