Q:

How to add Interactive Markers in URDF robot model

In the interactive markers tutorial, creation of the markers is made by creating simple objects (CUBES) in server  Interactive server program. How add interactive markers to robot saved as urdf model?

A:

You cannot describe interactive markers in the URDF, but when you're creating the interactive marker you can specify a link in the tf tree to which it is attached, and that link is described in the URDF.
