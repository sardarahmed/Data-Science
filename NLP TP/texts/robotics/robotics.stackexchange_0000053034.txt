A:

ros_lib needs to be in the /libraries folder of the sketchbook folder.  Did you put it there?  Seems like it may not be seeing ros.h
If you try to import the library from the arduino IDE menu, does the ros_lib folder show up in the list?  If not, then it is not in the correct location.
