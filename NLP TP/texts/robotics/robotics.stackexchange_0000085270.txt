you are missing a package in your system. You have not installed the package that provides gazebo_ros_utils.
The package that contains that is ros-indigo-gazebo-plugins. You should install everything ROS + Gazebo before trying to use the simulations. That is why we suggest you launch it at the RDS, because everything is configured.
