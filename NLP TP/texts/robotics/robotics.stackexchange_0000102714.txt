Q:

How can i remove this message from my terminal?

Hi. I had ROS 1(noetic) on my Ubuntu. Then i removed it and installed ROS 2. But whenever i open the terminal, i see this message and i don't know how to remove it. Could you help me with that? Thanks
bash: /opt/ros/noetic/setup.bash: No such file or directory

A:

I figured it out. This message's origin is ~/.bashrc file. You should type:
nano ~/.bashrc

After that go to the bottom and you will see
source /opt/ros/noetic/setup.bash

Delete this line of code. Then the message won't appear anymore.
